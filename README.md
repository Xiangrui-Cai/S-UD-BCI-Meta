脊髓损伤患者常常面临排尿和排便功能障碍，研究表明，大约有 70% 的中枢神经系统（CNS）疾病患者，如帕金森病、中风、多发性硬化或脊髓损伤患者，都会出现大便失禁和便秘的问题。这些状况对他们的生活质量与尊严造成了严重影响。
